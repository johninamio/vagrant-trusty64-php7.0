# vagrant-trusty64-php7.0
*Version 1.0-beta*

**Installation**  

    $ git clone https://github.com/johninamio/vagrant-trusty64-php7.0.git
    $ cd vagrant-trusty64-php7.0/ && vagrant up

**System**  

- Ubuntu 14.04
- Apache 2.4
- MySQL 5.5
- PHP 7.0
- phpmyadmin

**Utilities**  

- Composer
- GIT
- NodeJS, NPM
- Bower, Grunt, Gulp
